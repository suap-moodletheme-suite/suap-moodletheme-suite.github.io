Sobre o Projeto

Por que esta suíte?

O SUAP (Sistema Unificado de Administração Pública) é o sistema de gestão utilizado pelos Institutos Federais. Esta suíte garante que a experiência no Moodle seja visualmente coerente com o restante do ecossistema institucional.

Os componentes são desenvolvidos de forma independente e modular, permitindo adotar apenas o que for necessário para cada implantação.

Instalação

Como instalar

Instalação via git direto nos diretórios do Moodle, seguindo a convenção de nomenclatura de plugins.

1
Clone o tema principal
No diretório theme/ da sua instalação Moodle:
git clone https://github.com/suap-moodletheme-suite/moodle-theme_suap.git theme/suap
2
Adicione os blocos desejados
No diretório blocks/:
git clone https://github.com/suap-moodletheme-suite/moodle-block_course_gallery.git blocks/course_gallery
3
Execute as atualizações de banco
Acesse o painel de administração do Moodle e clique em Notificações para que os plugins sejam instalados automaticamente.
4
Ative o tema SUAP
Em Administração do site → Aparência → Temas, selecione o tema SUAP como padrão.
