Sobre o Projeto

Por que esta suíte?

O SUAP (Sistema Unificado de Administração Pública) é o sistema de gestão utilizado pelos Institutos Federais. Esta suíte garante que a experiência no Moodle seja visualmente coerente com o restante do ecossistema institucional.

Os componentes são desenvolvidos de forma independente e modular, permitindo adotar apenas o que for necessário para cada implantação.

Instalação

Como instalar

Os componentes da suíte seguem o padrão nativo de plugins do Moodle e podem ser instalados facilmente via interface web ou extração manual no servidor.

Via Interface Web (ZIP)

Recomendado para Administradores
1
Baixe o ZIP da Release
Acesse a aba Releases no repositório do plugin/tema desejado e baixe o arquivo .zip mais recente.
2
Upload no Moodle
Acesse Administração do site → Plugins → Instalar plugins e envie o arquivo .zip.
3
Conclua a Instalação
Siga as instruções em tela e execute as atualizações de banco de dados do Moodle.

Manual no Servidor (ZIP)

Extração em diretório
1
Extraia o Tema
Baixe a Release e extraia o conteúdo na pasta theme/suap.
2
Extraia os Blocos
Extraia os blocos desejados na pasta blocks/<nome_do_bloco> (ex: blocks/course_gallery).
3
Notificações e Ativação
Acesse Administração do site → Notificações para atualizar o banco de dados e ative em Aparência → Temas.
